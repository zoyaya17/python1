Flask Calculator App

Опис

Це проста Flask аплікація, яка дозволяє виконувати арифметичні операції через веб-інтерфейс.

Доступні маршрути

  • / — Головна сторінка. Виводить вітальне повідомлення з випадковим кольором.
  • /calculate/ — Форма для введення чисел a і b, а також вибору операції. Можливі операції:
    • Додавання (+)
    • Віднімання (-)
    • Множення (×)
    • Ділення (÷)
  • /col/ — Відображає випадковий колір у форматі rgb(r,g,b).
  • /doc/ — Відображає цей README.md у браузері.

Використання

1. Запустіть сервер: 

python app.py

2. Перейдіть у браузер і відкрийте: http://127.0.0.1:5000/

Приклад використання API

Додавання a = 5, b = 10

GET /calculate/?a=5&b=10&operation=sum

Відповідь:

<h1>SUM a & b = 15</h1>

Автор

Цю аплікацію створено для демонстрації можливостей Flask.
